## Running Locally

Clone the repository:

```bash
git clone https://github.com/derrick-scott/derrick-scott.github.io.git
```

Navigate to the project directory:

```bash
cd derrick-scott.github.io
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

## Deployment

This site is deployed using GitHub Pages.
